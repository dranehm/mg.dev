# mg.dev | Mhenard Galapate

A modern, animated portfolio website showcasing an IT professional's skills, experience, and certifications. Built with Next.js 16, Tailwind CSS, and Framer Motion for smooth animations.

## 🚀 Features

- **Responsive Design** – Works seamlessly on desktop, tablet, and mobile
- **Smooth Animations** – Framer Motion-powered scroll and entrance effects
- **Dark Mode Support** – Professional design with light/dark mode
- **Resume Download** – One-click resume PDF download
- **Contact Section** – Direct links to email, LinkedIn, and phone
- **Tech Stack Display** – Organized skill categories with icons

## 🛠 Tech Stack

- **Framework**: Next.js 16.2.0 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion 12
- **Icons**: react-icons (Lucide)
- **Language**: TypeScript

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dranehm/mg.dev.git
   cd mg.dev
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## 🏃 Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the live preview.

## 🏗 Building

Create an optimized production build:

```bash
npm run build
npm start
```

## 📝 Content Customization

Edit the portfolio content in `lib/data.ts`:

- **Profile Info** – Name, bio, contact details
- **Skills** – Organize tech stack by category
- **Experience** – Work history and roles
- **Certifications** – Training and achievements
- **Resume** – Replace `public/resume.pdf` with your actual resume

## 🚀 Deployment

Deploy easily to Vercel:

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or connect your GitHub repository to Vercel for automatic deployments on every push.
